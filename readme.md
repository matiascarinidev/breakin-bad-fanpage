# Breaking Bad - Fan Page

Proyecto de sitio web informativo sobre la serie Breaking Bad.

## Tecnologías

- HTML5
- CSS3 (flexbox, grid, animaciones)

## Estructura

- Index con secciones: inicio, sobre la serie, elenco, temporadas, curiosidades, contacto
- Páginas dedicadas: elenco.html, curiosidades.html, T1.html a T5.html
- Formulario de contacto con validación HTML
- Página de confirmación y error 404
